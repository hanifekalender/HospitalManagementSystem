# Hospital Management System

A web-based Hospital Management System developed using:

- Java
- JSP
- Servlets
- MySQL
- Maven
- Apache Tomcat

## Features

- User Login & Registration
- Dashboard Statistics
- Patient Management (CRUD)
- Doctor Management (CRUD)
- Appointment Management
- Medical Records
- Prescriptions
